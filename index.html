    01. About

    이 위키는 학습 과정에서 정리한 지식을 체계적으로 보관하는 공간입니다. Markdown이 아닌 HTML + JavaScript 기반으로 각 페이지를 직접 설계하고 관리합니다.

    Midnight & Gold 팔레트와 글래스 패널, 헤어라인 골드 스트로크로 프리미엄 기술 문서의 읽기 경험을 제공합니다.

    02. Getting Started

    새 문서는 document/ 디렉터리에 HTML 파일로 추가합니다.

    LOCAL_PREVIEW.SH 
    # Python 내장 서버로 미리보기
python -m http.server 8000

# 브라우저에서 열기
http://127.0.0.1:8000
    참고: 로컬 미리보기 시 정적 파일 서버가 필요합니다. Live Server 확장 또는 Python http.server를 사용하세요.
