    Wiki Documentation

    simigami Study Wiki의 문서 구조, 태그, 레벨, 메타데이터를 이해하기 위한 공식 가이드입니다.

    01. Overview

    이 위키는 태그 + 레벨 기반으로 문서를 분류합니다.

    각 위키 주제는 고유 설명 페이지 document/{name}.html을 가지며, Library에서 태그/검색으로 탐색할 수 있습니다.

    02. Tags

    태그는 문서의 주제 영역을 나타냅니다. 예: Computer Science, Algorithms.

    태그를 클릭하면 library.html#{tag-id}로 이동해 해당 태그 문서 목록을 확인합니다.

    태그 정의는 assets/data/wiki-taxonomy.json에서 중앙 관리합니다.

    03. Levels

    레벨은 문서의 거시적·미시적 범위를 숫자로 표현합니다.

    낮은 레벨은 넓고 거시적입니다. 예: Algorithms (Lv.2).

    높은 레벨은 구체적이고 미시적입니다. 예: Unreal Engine Dedicated Server, DFS (Lv.5).

    레벨 기반 Library 필터는 추후 제공 예정입니다.

    04. Document Metadata

    모든 문서 상단(page-header)에는 Level, Tags 메타 카드가 각각 별도 줄에 표시됩니다.

    • Level: 문서 레벨 (Lv.4)
    • Tags: 연결된 태그 목록

    05. Links

    문서 간 이동은 두 방식을 사용합니다.

    글자 링크: 본문 텍스트에서 바로 이동합니다.

    카드 링크: 대표 이미지, URL, 설명이 포함된 프리뷰 카드로 이동합니다.

    모든 링크는 새 페이지(새 탭)에서 열립니다.
